===============================================================================
Weather App
===============================================================================
Basic functionality weather app making
use of the Open Weather API available 
at http://openweathermap.org/
-------------------------------------------------------------------------------


Installation
-------------------------------------------------------------------------------
1.) Open the most recent release (Weather) using
IDE of choice
2.) Compile and load onto compatible device
3.) At this moment in time, the application will appear 
to crash upon launch. Select okay and continue.
4.) Application will request for permissions. Approve 
in order to continue. 


Usage
-------------------------------------------------------------------------------
Application will automatically load data from the API.
Swipe to update and city search will be available in 
future launches.
